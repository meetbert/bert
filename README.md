**BERT** is an AI-powered invoice processing platform. Each user is assigned a dedicated email address — vendors send invoices directly to that address, and BERT automatically receives the email, detects any attached or linked invoice, and runs it through a multi-agent pipeline to extract structured data: vendor, date, line items, totals, tax, and more. The extracted data is stored in a Supabase database and surfaced through an interactive dashboard where users can search, filter, and analyse their invoice history. BERT is designed to eliminate manual data entry and give businesses full visibility into their spending with zero effort.

To use BERT, visit the link above, log in with the demo credentials, and explore the dashboard. To test the pipeline, send an invoice email to the address shown in your account, or use the example invoices in the `example_inputs` folder.

## Dataset Used

BERT processes invoices received via email — no static dataset is required. No proprietary or third-party training datasets were used; our system uses open source models, so no training data was required. Example invoices used to test the system are included in the `example_inputs` folder.
